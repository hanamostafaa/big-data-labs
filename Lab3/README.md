# Running the Spark Notebook

## 1. Install Requirements

Install the required packages:

```bash
pip install pyspark jupyter
```

## 2. Open the Project Directory

Navigate to the project folder using the command prompt or terminal.

Example:

```bash
cd path/to/project
```

## 3. Start Jupyter Notebook

Run the following command:

```bash
jupyter notebook
```
if not working try
```bash
python -m jupyter notebook
```

## 4. Run the Notebook

## Notes

- Make sure **Java is installed**, as Spark requires Java.
